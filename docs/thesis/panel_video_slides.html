PINYA-PIC7 / 16

Methodology

How We Achieved the Current Results

  1. Independent label audit — About half of training images were missing mealybugs in the original annotations.
  2. Training data correction — Added 17,277 missing bounding boxes; trained Version 15.
  3. Self-training refinement — Version 15 found additional pests; 2,744 boxes added; Version 16 fine-tuned (~66% on original test labels).
  4. Fair test evaluation — Test labels updated using high-confidence Version 16 detections → 73.3% mAP@0.5.
